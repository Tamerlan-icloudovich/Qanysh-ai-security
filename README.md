# Qanysh-ai-security
Данный проект представляет собой мобильное приложение для обеспечения безопасности пользователей с использованием алгоритмов искусственного интеллекта (ИИ). Проект разработан в рамках дипломной работы и направлен на защиту от современных цифровых угроз, таких как фишинг, телефонное мошенничество, дипфейки (deepfakes) и манипуляции с медиафайлами.
